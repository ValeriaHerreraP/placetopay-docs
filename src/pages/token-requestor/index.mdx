# Beneficios de la tokenización

- **Mas seguridad para todas las transacciones y en su almacenamiento**.

- **Reducción de la tasa de declinación:** los tokens públicos tienen mayor visibilidad y nivel de seguridad con el emisor.

- **Mejor experiencia de usuario:** las transacciones no se declinaran por tarjeta vencida, ya que el token se actualizara automáticamente.

A continuación se describirán cada uno de los usos y los datos requeridos en la trama para cada uno de los casos. La respuesta para cada operación está unificada en un solo tipo de trama resultante.

Por defecto el lenguaje de las respuestas es en ingles.

### Servicios

Todos los consumos se realizan a la URL base, que será proporcionada por el área de operaciones y se le adiciona a dicha URL el endpoint del servicio a consumir.

Las muestras de API proporcionadas en este documento son solo para fines ilustrativos. En caso de confusión
entre la muestra y la especificación, la especificación debe considerarse la fuente real.

Las respuestas API exitosas devolverán un código de estado HTTP en el rango 2xx. El cuerpo de la respuesta (si
disponible) contendrá la respuesta de la API como se describe en este documento.
